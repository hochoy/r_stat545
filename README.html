Stat 545 and 547M Homework

by David Choy

Overview

As part of Dr. Jenny Bryan’s STAT545 and STAT547M data wrangling courses, we wrote R code and Rmd reports to present our understanding of R data wrangling and visualization. Homework 1-5 is part of STAT545 covering basic concepts of data readin, manipulation and plotting. Homework 6-10 is part of STAT547M covering data cleanup (munging)

Repo contents

  1. Homework 1 Learning R markdown and git
  2. Homework 2 Smelltesting gapminder data with base and ggplot2
  3. Homework 3 Finding trends in gapminder with dplyr and ggplot2
  4. Homework 4 Creating “split, apply, combine” functions to automate trend-finding in gapminder
  5. Homework 5 STAT545 wrapup: Order factor levels, choosing color schemes and writing figures to file
  6. Homework 7 Data munging real-life datasets with dplyr, tidyr, stringr and regular expressions: Halloween Candy
  7. Homework 8 Writing your own R package
  8. Homework 9 Automating a data analysis pipeline
  9. Homework 10 Data from the web
